Files included:

  oura_health-1_8_67.spl
    Splunk app package (tar.gz). Contains 6 Dashboard Studio dashboards
    (Today, Sleep, Heart Health, Activity, Wellness, Ring), navigation,
    saved searches (battery model, HR baseline, scheduled dedup maintenance),
    and static assets. Install via Splunk Web (Apps → Install app from file)
    or extract to $SPLUNK_HOME/etc/apps/.

  oura_to_hec_with_phi.py
    Oura Ring API → Splunk HEC ingest script. OAuth2 + PKCE authentication,
    per-type field filtering, checkpoint-based incremental sync, client-side
    dedup store with per-target tracking, multi-target HEC fan-out, HEC
    pre-flight token validation, and fcntl-based lock file to prevent
    concurrent runs. Requires Python 3.9+ and the requests library.

  INSTALL.md
    Complete installation and configuration guide covering Oura API setup,
    Splunk HEC configuration, OAuth2 authorization, environment variables,
    multi-target configuration, cron automation, log rotation, data reference,
    dedup store operation, scheduled dedup maintenance, checkpoint management,
    and troubleshooting.
